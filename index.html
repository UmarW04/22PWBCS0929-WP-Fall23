    Project Description

    The Online Bookstore Website is an e-commerce platform for buying and selling books online. It provides a user-friendly interface for customers to browse, search, and purchase books, as well as for sellers to list their books for sale. The website also includes features such as user reviews, ratings, and a secure payment system.

    Technologies Used:

    • Front-end: HTML, CSS, JavaScript
    • Back-end: PHP
    • Database: MySQL

      • Outcomes:
    • Users can create accounts, log in, and manage their profiles. Books are categorized by genre, author, and rating for easy navigation. A search functionality allows users to find specific books. Sellers can list books for sale, providing details like title, author, condition, and price. Users can read reviews and ratings from other customers before making a purchase. A secure payment gateway allows for online transactions. Admin panel for site management, user moderation, and book management.
